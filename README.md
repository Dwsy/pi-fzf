# pi-fzf

Fuzzy file autocomplete for [pi](https://github.com/mariozechner/pi) powered by fzf.

Type `@` followed by a fuzzy query to get ranked file suggestions in the editor.

## Requirements

```bash
brew install fzf fd
```

## Installation

```bash
pi install ~/.pi/agent/extensions/pi-fzf
```
